## Getting Started

### Prerequisites
```bash
- Node.js 18+ installed
- Monad RPC endpoint access
- NadFun API access (optional)
- Funded wallets with MON tokens
```

### Quick Setup
```bash
# Clone repository
git clone https://github.com/Tru3Bliss/monad-nadfun-trading-bundler-sniper-volume-bot.git

# Install dependencies
cd monad-nadfun-trading-bundler-sniper-volume-bot
npm install

# Configure environment
cp .env.example .env
# Edit .env with your settings

# Run bot
npm start
```

### Configuration
```javascript
{
  "mode": "sniper",              // sniper, volume, bundler, stealth
  "rpcUrl": "your-monad-rpc",
  "privateKeys": ["key1", "key2"],
  "slippage": 5,                 // percentage
  "gasMultiplier": 1.2,
  "stealthMode": true,           // enable bubble map bypass
  "volumeTarget": 100000         // USD volume target
}
```
